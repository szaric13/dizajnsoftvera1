#  Graff – Hierarchical Presentation Editor
This project is a desktop application that represents a structured presentation editor similar to PowerPoint, implemented using advanced object-oriented design principles and multiple design patterns.

The application allows users to create and manage hierarchical presentation projects consisting of workspaces, projects, presentations, and slides. Users can visually organize content through a tree structure and edit slide content within a tabbed interface.

Each project contains presentations that consist of slides. Slides support adding and manipulating different types of elements such as images, text, and custom graphic logos. Users can move, resize, rotate, zoom, and delete elements, providing full control over slide layout and content.

The system supports drag & drop functionality, multi-selection of elements, and dynamic UI updates. Changes in the project structure or slide content are automatically reflected in the interface through an Observer-based event system.

Additionally, the application includes a logging and notification system that handles errors, warnings, and system events, ensuring proper feedback during user interaction.
